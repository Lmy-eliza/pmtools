# 飞书消息规范

## 适用范围
所有发送的消息（包括飞书群消息、小窗消息）

## 称谓
### 飞书群消息
称"同学"，不称"同事"

### 小窗消息
- 通用规则：称呼中文名最后两个字
- 特殊情况：
  - 中文名只有两个字 -> 称呼全名（如"徐畅"）
  - 中文名有四个字 -> 称呼最后两个字（如"曹大林" -> "大林"）
  - 邮箱前缀 -> 转换为中文名后再取最后两个字（如 caodl -> "大林"）

## AI备注格式
所有消息末尾统一添加 `(本条消息由AI生成)`，连续放在语句末尾，不另起一行

## 特殊规则
- 如果收件人包含用户本人（李梦莹），则不发送消息给用户本人，避免重复通知

## 消息排版
- 减少不必要的空行
- 多点内容使用有序列表（如 1. 2. 3.）
- 结构化内容可使用无序列表
- 感谢类结束语可空一行后另起，如"谢谢大家配合"
- **涉及文档/资料时，必须附上对应链接**，禁止只说"详见文档"而不给链接
