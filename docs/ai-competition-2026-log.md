## 阶段四：Track 3 产品方案设计

**日期**：2026-05-14

### 4.1 产品定位

**赛题**：效能跃升·AI开挂  
**产品名**：smart-planner  
**核心定位**：整车项目计划智能助手，以知识沉淀为基础、级联影响为卖点、AI风险翻译为亮点

### 4.2 产品链路 — Eliza的4步法

这是 Eliza 提出的核心产品思路，CC 在此基础上细化：

```
Step 1: 知识沉淀          Step 2: 标准模板         Step 3: 计划生成         Step 4: 风险识别
┌──────────────┐     ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│ 历史项目计划  │ ──→ │ 分车型制定    │ ──→ │ 输入SOP日期   │ ──→ │ 计划调整后    │
│ 画板→结构化   │     │ 最标准的计划  │     │ 自动生成建议版 │     │ AI识别风险    │
│ WBS数据录入   │     │ 全新/改款/海外│     │ 阀点+节点自动排│     │ 级联影响分析  │
└──────────────┘     └──────────────┘     └──────────────┘     │ 措施建议输出  │
                                                               └──────────────┘
```

### 4.3 评分策略对照

| 评分维度 | 权重 | 我们的打法 | 目标分 |
|---------|------|-----------|--------|
| 实用性 | 40% | 来自计划组真实工作流，量化痛点（单项目排期8人天），闭环Demo | 30+/40 |
| 平权性 | 30% | 所有车型项目组都能复用，知识库模板化=低门槛部署 | 22+/30 |
| 创意性 | 30% | 知识沉淀+模板推理+级联影响+风险翻译=多步AI pipeline | 22+/30 |

### 4.4 技术架构

**架构思路**：Skill+Web 双层协作，退化为纯 Web

```
飞书Skill/Agent ──生成──→ Web应用（可视化+精调）──确认──→ 飞书文档输出（画板呈现）
                         │ 甘特图/时间线/节点编辑  │
                         │ 级联影响分析+风险仪表盘  │
                         │ AI风险识别+建议          │
```

**技术栈**：
- Web前端：React + TypeScript + Vite
- UI组件：Ant Design
- 图表可视化：ECharts
- AI对话：阿里云百炼 / Claude API
- 数据层：本地JSON/localStorage + Excel导入导出
- 飞书集成：lark-cli（Skill层）+ 飞书画板输出
- 部署：Netlify

**退化方案**：如果飞书集成复杂度过高，Web应用独立运行，计划以Excel导出

### 4.5 核心功能定义

| 功能模块 | 优先级 | 描述 |
|---------|-------|------|
| WBS知识库 | P0 | Excel导入历史项目WBS数据；可视化展示WBS层级结构 |
| 标准模板管理 | P0 | 按车型维护标准WBS模板；支持切换查看 |
| 对话式计划生成 | P0 | 输入车型+SOP日期，自动倒排生成项目计划表 |
| 级联影响分析 | P0 | 修改某节点→自动计算下游所有节点变化+变更对比表 |
| AI风险识别 | P0 | 对比调整后计划vs标准计划，识别风险并给建议 |
| 项目健康度评分 | P1 | 基于延期/偏差等维度计算综合健康度分数 |
| 风险报告生成 | P1 | 一键生成结构化风险总结报告 |

### 4.6 领域知识（复用自 QoderWork PRD）

- **阀点体系**：G5（项目启动）→ G4（方案冻结）→ G3（设计冻结）→ G2（工程冻结）→ G1（量产准备）→ G0（SOP）
- **WBS三级结构**：L1项目阶段 / L2工作包 / L3任务节点
- **车型-周期映射**：全新车型36月 / 小改款12月 / 海外项目10月
- **时间倒排规则**：以SOP为终点，按车型默认周期倒推各阶段起始时间
- **非工作日顺延**：计算日期若落在周末/节假日，自动顺延至最近工作日

### 4.8 演示场景设计

1. **知识沉淀**（1min）：上传历史项目Excel → 解析为WBS → 展示知识库
2. **模板查看**（30s）：切换全新/改款/海外模板，展示标准阀点
3. **生成计划**（2min）：输入"全新车型，SOP 2028-06-01" → 自动倒排 → 甘特图
4. **级联调整**（2min）：G5延后15天 → 系统自动重算下游 → Before/After对比
