# AI五大能力分类说明

<callout emoji="bulb" background-color="light-blue">

**能力分类逻辑**：基于AI技术本质能力（而非业务动作）进行抽象，确保跨Agent复用、跨管理域适配。每个业务活动节点标注其涉及的能力编号，便于统计AI渗透率和能力覆盖度。

</callout>

| 能力维度 | 定义 | 典型应用场景 |
|---------|------|-------------|
| ①智能生成 | 自动生成文档、材料、计划、报告等交付物，提升交付效率 | 任务书自动生成、项目计划编排、会议纪要生成、数据看板生成 |
| ②智能审核 | 对人工交付物进行合规性、完整性、一致性校验，提供审核建议 | 风险提案预审、材料完整性校验、目标对齐合理性检查 |
| ③风险预警 | 基于数据分析进行风险识别与主动预警，监控流程执行偏差 | 进度偏差预警、风险等级判定、超期催办、阈值触发通知 |
| ④知识检索 | 从知识库/历史数据中检索经验并辅助决策，沉淀组织智慧 | 经验总结入库、模板推荐、历史案例检索、最佳实践推送 |
| ⑤数据洞察 | 多维数据采集、计算与可视化分析，支撑管理决策 | 目标达成率分析、决策效率统计、专业排名看板、趋势预测 |
