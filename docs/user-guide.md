# Smart Planner 使用说明

> **版本**：v1.0（2026-05-31）
> **在线体验**：https://project-planner-pmtools.netlify.app/?mode=chatpanel

---

## 产品定位

Smart Planner 是面向整车研发的**项目计划智能助手**。用户通过自然语言对话，即可自动生成包含阀点、泳道、活动节点的完整项目计划，并在可视化画布上查看和编辑。

**核心场景**：项目启动前的计划编制。输入车型规模和 SOP 目标日期，AI 基于 532 条标准活动模板自动倒推排期，将 8 人天的手工排期工作缩短到 1 小时以内。

## 场景 1：AI 对话生成计划

**适用场景**：从零开始创建一份新的项目计划。

### 操作步骤

1. **打开 Smart Planner**，点击顶部工具栏右侧的「项目规划师」按钮（💬 图标），打开 AI 对话面板

2. **输入项目信息**，例如：
   ```
   生成一个全新车型的项目计划，SOP 2028-06-01
   ```

3. **AI 确认参数**
   - 如果信息充足（给出了车型规模），AI 会直接生成计划
   - 如果信息不足，AI 会一次性列出需要确认的参数：
     - 车型规模（全新 / 中改 / 小改 / 海外 / 平台首发）
     - SOP 日期
     - 是否有阀点裁剪需求
     - 是否有特殊活动关联关系（FF/FS/SS/SF）
     - 关键路径是否有指定需求
     - G 阀时间是否需要避开特定日期

4. **AI 生成 JSON**，回复中包含完整的 ProjectData JSON

5. **点击「导入计划」按钮**，计划自动渲染到画布上

### 支持的车型规模

| 规模 | 阀点数 | 标准活动数 | 总节点数 |
|------|--------|-----------|---------|
| 平台首发 | 12 | 446 | 458 |
| 全新 | 12 | 84 | 96 |
| 中改 | 7 | — | — |
| 小改 | 5 | — | — |
| 海外 | 6 | — | — |

### 试试这些话术

- `排个 P7+ 中改的计划，SOP 2027-09-01`
- `帮我规划一个海外车型项目`
- `生成全新车型计划，去掉 G9 和 G2 阀点`

## 场景 2：画布浏览与检查

**适用场景**：查看 AI 生成的计划，确认整体结构和时间安排。

### 画布操作

| 操作 | 方式 |
|------|------|
| 缩放 | 工具栏缩放控件，或鼠标滚轮 |
| 平移 | 水平/垂直滚动条 |
| 切换时间粒度 | 工具栏「日/周/月/季度」按钮 |
| 查看节点信息 | 点击节点，右侧显示属性面板 |
| 折叠泳道 | 点击泳道名称左侧的折叠箭头 |

### 用 AI 做快速检查

在对话面板中输入：

- `当前计划概况` → AI 汇总节点数、泳道数、周期等基础信息
- `帮我检查一下这个计划` → AI 按 7 个维度诊断（阀点完整性、时序、间距、泳道覆盖、活动密度、时间合理性、工作日检查）
- `G5 到 G4 之间多少天？` → AI 直接计算并回答

## 场景 3：计划微调（小改动）

**适用场景**：需要调整 1-2 个节点的时间、名称或属性。**直接在画布上操作比对话更高效。**

### 画布编辑操作

| 操作 | 方式 |
|------|------|
| 调整时间 | 拖拽节点到新位置，顶部实时显示对应日期 |
| 修改名称 | 双击节点，在属性面板编辑名称 |
| 修改颜色 | 选中节点，在属性面板修改颜色 |
| 调整工期 | 选中矩形节点，拖拽右侧边框调整宽度 |
| 删除节点 | 选中节点，按 `Delete` 或 `Backspace` |
| 添加节点 | 工具栏选择图形类型（◇ △ ▭），在画布上点击放置 |
| 添加连接线 | 按 `L` 切换到连线工具，依次点击起始和目标节点 |
| 添加泳道 | 工具栏 `+` 按钮，新泳道添加到最下方 |
| 泳道排序 | 在左侧面板拖拽泳道名称调整顺序 |
| 泳道重命名 | 双击左侧泳道名称直接编辑 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `V` | 选择工具 |
| `D` | 添加菱形节点（里程碑） |
| `T` | 添加三角形节点（决策点） |
| `R` | 添加矩形活动 |
| `L` | 连线工具 |
| `Delete` / `Backspace` | 删除选中项 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Shift + Z` | 重做 |
| `Ctrl + S` | 保存项目 |
| `Ctrl + E` | 导出 DrawIO |
| `Escape` | 取消选择 / 退出当前工具 |
| `?` 或 `F1` | 打开帮助面板 |

## 场景 4：AI 对话迭代（大改动）

**适用场景**：需要批量修改、涉及多个节点联动、或者需要 AI 帮助判断影响范围。

### 推荐使用 AI 对话的情况

- 需要同时移动多个阀点或活动
- 需要新增/删除多个活动节点
- 涉及活动间的依赖关系（如 A 完成后 B 才能开始）
- 想让 AI 分析某个变更的影响范围
- 想让 AI 帮某个泳道补充典型活动

### 示例对话

**时间调整**：
```
把 G5 延后 15 天
```
AI 会修改 G5 的日期，并在回复中说明哪些关联活动可能受影响（但不会自动级联修改其他节点）。

**批量操作**：
```
给测试验证泳道补充几个典型活动
```
AI 会根据泳道类别推荐 2-4 个典型活动，自动对齐到相关阀点区间。

**关联关系**：
```
把 VR1.0 软件发布和 SR1.0 软件发布设为 FF 关系（同时完成）
```
AI 理解 FF/FS/SS/SF 等项目管理术语，会用连接线样式做可视化标注。

**风险分析**：
```
如果 G5 延后 15 天，哪些活动会受影响？
```
AI 分析级联影响范围，列出直接影响的活动和下游挤压效应。

### 编辑后的操作

AI 每次编辑后都会输出完整的修改后 JSON。点击「导入计划」按钮即可将修改后的计划覆盖到画布上。

**注意**：AI 遵循「单点编辑」原则——只修改你明确指定的目标节点，不会自动级联修改其他节点。AI 会在回复中列出可能受影响但未自动调整的关联节点，由你决定是否继续调整。

## 场景 5：版本回退

**适用场景**：对修改结果不满意，想回到之前的状态。

### 三种回退方式

| 方式 | 操作 | 适用场景 |
|------|------|---------|
| **撤销** | 工具栏 ↩️ 按钮或 `Ctrl + Z` | 回退最近的操作，最多 50 步 |
| **重做** | 工具栏 ↪️ 按钮或 `Ctrl + Shift + Z` | 撤销过头了，恢复操作 |
| **版本快照** | 右侧面板 → 版本历史 → 选择版本恢复 | 回到某个命名保存的历史版本 |

### 建议

- 在 AI 生成计划并导入后，**先保存一个版本快照**（取名如"AI初版"），方便后续随时回退
- 画布上的手动编辑支持逐步撤销（最多 50 步）
- 如果在对话中让 AI 做了大量修改，觉得不满意，可以直接在对话中说"还原到上一个版本"，AI 会提示你使用撤销按钮

## 场景 6：导出与分享

**适用场景**：计划确认后，需要导出给他人查看或导入到其他工具。

### 导出方式

| 方式 | 操作 | 输出格式 | 用途 |
|------|------|---------|------|
| **导出 JSON** | 工具栏 → 更多菜单（⋯）→ 导出 JSON | `.json` 文件 | 给其他 AI Agent 或系统读取 |
| **导出 DrawIO** | 工具栏 → 导出按钮（⬇） | `.drawio` 文件 | 导入飞书画板或 draw.io 继续编辑 |
| **复制为图片** | 工具栏 → 📷 按钮 | PNG（剪贴板） | 粘贴到文档、邮件、聊天中 |
| **JSON 粘贴导入** | 工具栏 → 更多菜单（⋯）→ 粘贴 JSON | — | 从其他来源导入计划 |

### 文件上传

在 AI 对话面板中，点击 📎 按钮可上传文件作为对话上下文：
- 支持 `.md` `.txt` `.json` `.csv` 格式
- 文件大小上限 50KB
- 文件内容会作为消息的一部分发送给 AI

## 场景 7：项目管理

**适用场景**：管理多个项目、切换项目、查看项目列表。

| 操作 | 方式 |
|------|------|
| 查看项目列表 | 工具栏 → 项目列表面板 |
| 新建项目 | 导入新的 JSON 即创建新项目 |
| 切换项目 | 项目列表中点击要打开的项目 |
| 项目设置 | 工具栏 → 设置按钮 → 修改日期范围、时间间隔等 |

**数据存储**：
- 所有项目数据自动保存在浏览器的 IndexedDB 中（关闭浏览器不丢失）
- 自动保存频率：编辑后 2 秒自动保存，关闭页面时强制保存
- 支持飞书云端同步（需登录飞书账号）

## 关于 AI 对话的注意事项

### AI 能做的

- 生成完整的项目计划（阀点 + 泳道 + 活动节点）
- 编辑现有计划（调时间、增删节点、改属性）
- 查询计划信息（统计、间距计算、概况）
- 诊断计划合理性（阀点完整性、时序、间距、空泳道等）
- 智能补充活动（根据泳道类别推荐典型活动）
- 理解项目管理术语（FF/FS/SS/SF 依赖关系）

### AI 暂不支持的

- 解析飞书文档或飞书表格（需通过 CLI 工具处理后导入）
- 自动级联修改（修改一个阀点后自动调整所有关联活动）
- 实时流式输出（当前为完整响应后一次性返回）
- 跨会话记忆（刷新页面后对话记录清空，但画布数据保留）

### 最佳实践

1. **先生成后微调**：让 AI 先快速生成一个初版，看到具体计划后再提修改需求
2. **小改用画布**：单个节点的时间/名称/颜色调整，直接在画布上拖拽或双击编辑更快
3. **大改用对话**：批量修改、依赖关系、影响分析等复杂操作交给 AI
4. **及时保存版本**：重要修改前保存版本快照，方便回退
5. **一次说清**：给 AI 足够的信息（车型规模 + SOP + 特殊需求），减少来回追问

## 技术限制说明

| 限制 | 说明 |
|------|------|
| 大赛演示版 | 当前为大赛作品演示版，请勿使用真实敏感数据 |
| AI 响应时间 | 生成完整计划约需 5-15 秒，视计划规模和网络状况而定 |
| 节点规模 | 推荐 100 节点以内保持最佳流畅度，96 节点已验证流畅 |
| 浏览器支持 | 推荐 Chrome / Edge 最新版本 |
