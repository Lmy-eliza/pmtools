# Project Planner 可视化项目规划工具

> 拖拽式项目管理器，让项目排期效率提升 10 倍

## 核心功能

- **拖拽即排期**：拖动节点即可调整日期，依赖节点自动同步更新
- **关键路径高亮**：自动计算并高亮项目关键路径，掌控进度风险
- **飞书导出**：支持导出为飞书画板（draw.io 格式），无缝衔接团队协作
- **本地存储**：数据保存在浏览器 localStorage，无隐私泄露风险
- **版本管理**：支持项目版本历史保存，随时回溯

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用场景

1. **项目排期**：用甘特图思维做可视化排期，拖拽调整直观高效
2. **进度汇报**：导出飞书画板，一键同步到团队文档
3. **进度复盘**：关键路径自动计算，快速定位延误风险点

## 技术栈

- React 19 + TypeScript
- Konva Canvas 图形渲染
- Zustand 状态管理
- Dexie 本地数据库
- TailwindCSS 样式
